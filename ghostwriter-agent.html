Process

How It Works

Ghostwriter follows a Think → Act → Verify loop inside an isolated sandbox, ensuring every fix is validated before it ever touches your codebase.

01

Detect

Listens for GitHub webhook events on PR open or sync. Spawns an ephemeral Docker sandbox to clone and test the branch.

02

Analyze

LLM reads test failure logs, queries relevant source files using agentic RAG, and reasons about the root cause using chain-of-thought.

03

Heal

Applies the code fix inside Docker, re-runs tests to verify, then pushes a self-healing commit or opens a refined Pull Request via Octokit.

Features

Production-grade autonomous healing with enterprise-level security and observability.

🧠

Agentic Reasoning

Chain-of-Thought prompting with semantic file retrieval to pinpoint exactly which code to fix.

🛡️

Zero-Trust Security

All AI-generated code runs in ephemeral Docker containers — the host is never exposed.

🔄

Auto Recovery

Patches syntax errors, null checks, dependency mismatches — then verifies before pushing.
